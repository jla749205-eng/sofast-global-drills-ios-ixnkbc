## Quick Reference üöÄ

### The 3 Commands You Need:
```bash
npm install -g eas-cli
eas build --platform ios --profile production
eas submit --platform ios --profile production
```

### The 3 Things You Must Do:
1. Update `eas.json` with your Apple credentials
2. Create app listing in App Store Connect
3. Submit for review

### The 3 Files You Must Have:
1. Privacy Policy (live at https://tmsofast.com/privacy)
2. Screenshots (3-5 images of your app)
3. App Description (see templates in guides)
