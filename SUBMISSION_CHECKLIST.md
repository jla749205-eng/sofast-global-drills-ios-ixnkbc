### 2. Capture iPad Screenshot (15 minutes)

**Required Dimensions:** 2048 x 2732 pixels (13-inch iPad Pro, portrait)

**Steps:**
1. Open Xcode
2. Window ‚Üí Devices and Simulators
3. Select "iPad Pro (13-inch)"
4. Run your app
5. Navigate to home screen
6. Press `Cmd + S` to save screenshot
7. Screenshot saves to Desktop
8. Verify dimensions (Right-click ‚Üí Get Info)

**Screens to Capture:**
- Home screen with drill list
- Drill details page
- Camera view during recording
- Results screen with scores
- Leaderboard rankings

**Tip:** Capture 3-5 different screens to showcase your app's features

### 3. Capture iPhone Screenshot (15 minutes)

**Required Dimensions:** 1320 x 2868 pixels (iPhone 16 Pro Max, portrait)

**Steps:**
1. In Xcode, select "iPhone 16 Pro Max" simulator
2. Run your app
3. Press `Cmd + S` to save screenshots
4. Capture the same screens as iPad

### 4. Join Apple Developer Program ($99/year)

1. Go to developer.apple.com
2. Click "Enroll"
3. Pay $99 annual fee
4. Wait for approval (24-48 hours)

### 5. Upload to App Store Connect

1. Log in to appstoreconnect.apple.com
2. Create new app
3. Upload screenshots:
   - iPad Pro (13-inch): 2048 x 2732
   - iPhone 16 Pro Max: 1320 x 2868
4. Enter URLs:
   - Support URL: https://tmsofast.com/support
   - Privacy Policy: https://tmsofast.com/privacy
5. Fill in app information
6. Build and upload app binary
7. Submit for review
